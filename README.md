# PUBG_Project

## Prerequisites
- R
- Rmd packages
- MAC OS

## Installation 
* Clone the repo:
``` 
git clone https://github.com/msouppe/PUBG_Project.git
```

* Navigate to `main_PUBG.R`

* Run the program:
```bash
Rscript main_PUBG.R 
```

## Project Structure
* `main_PUBG.R`: Main program
* `util.R`: Processes data and analyzer of data
